# Variants Dashboard

Interactive launch report for the Variant Grouping auto-grouping model — June 2026.

## Live page

**[variant_grouping_model_analysis.html](https://necmettintamgueney.github.io/variants-dashboard/variant_grouping_model_analysis.html)**

### Tabs

| Tab | Description |
|---|---|
| Executive Summary | Top-line KPIs, platform breakdown, quality signal |
| Detail: Overview | Full scale and quality metrics |
| What went wrong | Root cause analysis for the 55 corrected groups |
| By category | Edit rate heatmap by product category |
| All 55 edited groups | Filterable table of every corrected group |
| Products moved | Product-level correction log |
| Review coverage | Editor coverage by platform |
| QS OS Learn | Learn Template — hypothesis, metrics, qualitative feedback, learnings, decisions |
